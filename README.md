# 🚀 v5marketing

Sistema web desenvolvido em ASP.NET Core MVC para gestão de clientes e usuários.

## 📌 Funcionalidades

- 🔐 Login com autenticação via banco
- 👥 Cadastro de usuários com perfis (Administrador / Operador)
- 📊 Dashboard com gráfico (Chart.js)
- 📋 CRUD completo de clientes
- 🔎 Paginação, busca e ordenação
- 🌙 Modo escuro
- 📱 Responsivo (mobile)
- 🔔 Toasts de feedback
- 🛡️ Controle de permissões por perfil
- 📜 Auditoria de ações do sistema

## 🧱 Tecnologias

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5
- Chart.js

## 🧠 Aprendizados

Projeto focado em:

- arquitetura MVC
- autenticação e autorização
- boas práticas de CRUD
- experiência do usuário (UX)
- estrutura para sistemas reais
